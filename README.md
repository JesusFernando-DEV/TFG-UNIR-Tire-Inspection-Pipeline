# Tire Inspection Pipeline - TFG UNIR 🚗🔍

## 🎯 Overview
Final Degree Project (TFG) for UNIR Computer Engineering: **Automatic tire quality inspection** using **classical computer vision**. Detects **cracks** and reads **DOT codes** on sidewalls.

**Key features:**
- **Pipeline**: Canny edges → Hough lines → Harris corners → Tesseract OCR
- **Dataset**: TyreNet (1,698 real-world images)
- **Metrics**: >85% mAP, robust to dirt/lighting
- **Tech**: Python, OpenCV, Visual Studio Code, Streamlit demo
